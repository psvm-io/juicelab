JuiceLab
======
**JuiceLab** is a docker compose script that deploys a pentesting environment for the OWASP Juice Shop web application.

## Install
```
$ cd juicelab
$ docker compose up -d
```

## Content
* OWASP Juice Shop
* Kali Linux (Rolling)
